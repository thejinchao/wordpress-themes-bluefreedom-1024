wordpress-themes-bluefreedom-1024
=================================

这是一个WordPress主题，基于BlueFreedom开发。所做的调整是把宽度有724调整到了1024，并且提供了相应的title图片
安装方法：
	1. 首先安装Blue Freedom主题，网址 http://wordpress.org/themes/bluefreedom
	2. 按照Child Theme的方法，将这个调整后的主题安装到 wordpress的theme目录，并激活

参考：
	Child Theme : http://codex.wordpress.org/Child_Themes
